13.1 Contact Page

This page uses "Page with Sidebar" template you can find inside the "Page Attributes". There is also a featured image you can include but you don't have to. If you don't want to include featured image the page and the content will adjust.

The contact form itself is built using Jetpack Contact Form but you are free to use any other working contact form.

13.2 Right Content Block - Right Sidebar

Contact page is using the same "Right Sidebar" like the news page and the single post page. Using feature "Visibility" inside the "Right Sidebar" you can add widgets that will show up just on the contact page and other widgets that will show up on the news page.
