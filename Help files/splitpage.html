6.1. About Wines Page

To set up the About Me page you need to create a new page, you can do that by navigating to Pages > Add New.

You can give this page a title yet you do not have to include any content. Choose "Split Page - About Wines" from the Page Attributes section and click "Publish".

Split page is created using child pages. It contains content of the two child pages. That means that About Wines Page is a Parent Page and contains "Red Wine" and "White Wine" page content.

Open two different pages (in this case Red Wine and White Wine, you can name them any way you want) and inside the "Page Attributes" set - Parent: About Wines.
