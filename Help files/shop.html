11.1 How to set up shop products on your page?

Before you start with setting up your shop site, you need to install WooCommerce plugin: http://wordpress.org/plugins/woocommerce/.

As this isn't a WooCommerce theme but theme that supports this plugin and it is styled for it we will not explain how WooCommerce plugin is working. About setting up your shop and using WooCommerce plugin, please read more about it here: http://www.woothemes.com/woocommerce/
